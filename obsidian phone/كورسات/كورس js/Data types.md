### أنواع البيانات (Data Types) في جافا سكريبت

جافا سكريبت تدعم عدة أنواع من البيانات، والتي يمكن تصنيفها إلى نوعين رئيسيين: **أنواع البيانات البدائية** (Primitive Data Types) و**أنواع البيانات الكائنية** (Object Data Types). إليك شرحًا لكل نوع:

#### 1. أنواع البيانات البدائية

هذه الأنواع تمثل قيمًا بسيطة وغير قابلة للتغيير:

- **String**: تمثل سلسلة من الأحرف.
  ```javascript
  let name = "علي";
  ```

- **Number**: تمثل الأرقام، سواء كانت صحيحة أو عشرية.
  ```javascript
  let age = 25;
  let price = 19.99;
  ```

- **Boolean**: تمثل القيم المنطقية، إما `true` أو `false`.
  ```javascript
  let isActive = true;
  ```

- **Undefined**: تمثل قيمة غير معرفة، تُستخدم عندما لا يتم تعيين قيمة لمتغير.
  ```javascript
  let x;
  console.log(x); // undefined
  ```

- **Null**: تمثل قيمة فارغة أو غير موجودة.
  ```javascript
  let y = null;
  ```

- **Symbol**: تمثل قيمة فريدة وغير قابلة للتغيير، تُستخدم كمعرفات فريدة.
  ```javascript
  const uniqueID = Symbol('id');
  ```

- **BigInt**: تمثل أعدادًا صحيحة كبيرة جدًا، أكبر من ما يمكن أن يُخزن في نوع `Number`.
  ```javascript
  let bigNumber = BigInt(123456789012345678901234567890);
  ```

#### 2. أنواع البيانات الكائنية

تُمثل مجموعة من القيم، ويمكن أن تحتوي على أنواع بيانات أخرى:

- **Object**: تمثل مجموعة من الخصائص (key-value pairs).
  ```javascript
  let person = {
    name: "علي",
    age: 25
  };
  ```

- **Array**: نوع خاص من الكائنات يُستخدم لتخزين قائمة مرتبة من العناصر.
  ```javascript
  let fruits = ["تفاح", "موز", "برتقال"];
  ```

- **Function**: تعتبر دالة كائنًا أيضًا، ويمكن تعريفها وتخزينها كقيمة.
  ```javascript
  function greet() {
    console.log("مرحبًا!");
  }
  ```

### خلاصة

أنواع البيانات في جافا سكريبت تُساعد على تنظيم المعلومات والتعامل معها بشكل فعال. ففهم هذه الأنواع يُعتبر أمرًا أساسيًا لتطوير تطبيقات ويب ناجحة.
