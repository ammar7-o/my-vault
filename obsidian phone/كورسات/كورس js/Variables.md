### أنواع المتغيرات في جافا سكريبت للمبتدئين

في جافا سكريبت، يمكنك استخدام ثلاثة كلمات رئيسية لتحديد المتغيرات: **var**، **let**، و**const**. كل منها له خصائصه واستخداماته. دعنا نستعرضها بالتفصيل مع تقديم نصائح حول كيفية تسمية المتغيرات.

#### 1. `var`

- **الوصف**: تُستخدم لتعريف متغيرات يمكن تغيير قيمتها.
- **نطاق الرؤية**: يمكن استخدام المتغير في أي مكان داخل الدالة، أو عالميًا إذا تم تعريفه خارج أي دالة.
- **إعادة التعريف**: يمكنك إعادة تعريف المتغير في نفس النطاق.

**مثال**:
```javascript
var age = 25;
console.log(age); // 25

if (true) {
  var age = 30; // إعادة تعريف المتغير
  console.log(age); // 30
}

console.log(age); // 30
```

#### 2. `let`

- **الوصف**: تُستخدم لتعريف متغيرات يمكن تغيير قيمتها، لكن لها نطاق رؤية كتلي.
- **نطاق الرؤية**: المتغير مُتاح فقط داخل الكتلة التي تم تعريفه فيها (مثل الحلقات أو الجمل الشرطية).
- **إعادة التعريف**: لا يمكنك إعادة تعريف المتغير في نفس النطاق.

**مثال**:
```javascript
let score = 100;
console.log(score); // 100

if (true) {
  let score = 200; // متغير جديد
  console.log(score); // 200
}

console.log(score); // 100
```

#### 3. `const`

- **الوصف**: تُستخدم لتعريف متغيرات ثابتة، مما يعني أنك لا تستطيع تغيير قيمتها بعد تعريفها.
- **نطاق الرؤية**: مثل `let`، المتغير مُتاح فقط داخل الكتلة التي تم تعريفه فيها.
- **إعادة التعريف**: لا يمكنك إعادة تعريف المتغير أو تغيير قيمته.

**مثال**:
```javascript
const pi = 3.14;
// pi = 3.14159; // سيؤدي إلى خطأ

if (true) {
  const pi = 3.14159; // متغير جديد
  console.log(pi); // 3.14159
}

console.log(pi); // 3.14
```

### كيفية تسمية المتغيرات

عند تسمية المتغيرات، يجب أن تتبع بعض القواعد والنصائح:

1. **استخدام أحرف واضحة**: اختر أسماء تعبر عن محتوى المتغير. على سبيل المثال:
   - `age` (العمر)
   - `name` (الاسم)
   - `totalScore` (النتيجة الكلية)

2. **استخدام camelCase**: يُفضل استخدام نمط camelCase، حيث تبدأ الكلمة الأولى بحرف صغير وتبدأ الكلمات التالية بحرف كبير:
   - `firstName`
   - `totalAmount`

3. **تجنب الأرقام في البداية**: لا يمكن أن تبدأ أسماء المتغيرات برقم. على سبيل المثال، `1stPlace` غير صحيح.

4. **تجنب الكلمات المحجوزة**: لا تستخدم الكلمات التي لها معنى خاص في جافا سكريبت، مثل `if` أو `function`.

5. **اجعلها وصفية**: حاول أن تجعل اسم المتغير يعبر عن محتواه بوضوح، مثل `userAge` بدلاً من `x`.

### خلاصة

اختيار نوع المتغير المناسب وتسمية المتغيرات بطريقة واضحة يساعد في كتابة شيفرة برمجية منظمة وسهلة الفهم. يُفضل استخدام `let` و`const` في معظم الحالات، حيث يوفران تحكمًا أفضل في نطاق الرؤية.
