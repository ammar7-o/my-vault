### مقدمة للجافا سكريبت

جافا سكريبت هي لغة برمجة ديناميكية تُستخدم بشكل رئيسي لتطوير الويب. تم تصميمها لتعمل على جانب العميل، مما يعني أنها تُنفذ مباشرة في متصفح المستخدم. تُستخدم جافا سكريبت لإضافة التفاعلية إلى صفحات الويب، مثل التحقق من صحة النماذج، إنشاء الرسوم المتحركة، وتحديث المحتوى بدون الحاجة إلى إعادة تحميل الصفحة. 

تتميز جافا سكريبت بمرونتها، حيث يمكن استخدامها في تطوير التطبيقات الكبيرة والمعقدة باستخدام مكتبات وإطارات عمل مثل React وAngular وVue.js.

### تعريف مبسط للإيكم سكربت

الإيكم سكربت (ECMAScript) هو معيار يُحدد كيفية عمل جافا سكريبت. تم تطويره من قبل منظمة ECMA الدولية ويُعتبر الأساس الذي تبنى عليه جافا سكريبت. يحتوي الإيكم سكربت على مجموعة من القواعد والتوجيهات التي تحدد كيفية كتابة الشيفرة البرمجية، مما يضمن التوافق بين مختلف المتصفحات والأدوات التي تدعم جافا سكريبت.

ببساطة، يمكن اعتبار الإيكم سكربت بمثابة "قانون" لجافا سكريبت، حيث يُحدد ما يمكن وما لا يمكن أن تفعله اللغة.
### إيكم سكربت 6 (ECMAScript 6)

إيكم سكربت 6، المعروف أيضًا باسم ES6 أو ECMAScript 2015، هو إصدار رئيسي من معيار إيكم سكربت. تم إصداره في عام 2015 ويقدم مجموعة من الميزات الجديدة والتحسينات التي تجعل كتابة الشيفرة البرمجية أسهل وأكثر قوة. إليك بعض الميزات الرئيسية التي تم تقديمها في ES6:

1. **المتغيرات**: 
   - **let** و **const**: لتحديد المتغيرات. `let` يسمح بتعريف متغير يمكن تغييره، بينما `const` يستخدم لتعريف متغير ثابت.

2. **الدوال السهمية**:
   - توفر طريقة أقصر لكتابة الدوال، مثل:
     ```javascript
     const add = (a, b) => a + b;
     ```

3. **القوالب النصية**:
   - تتيح إنشاء نصوص متعددة الأسطر واستخدام تعبيرات داخل النصوص باستخدام علامة `` ` ``:
     ```javascript
     const name = "علي";
     const greeting = `مرحبًا، ${name}!`;
     ```

4. **الصفوف (Classes)**:
   - دعم البرمجة الكائنية بشكل أفضل مع تعريف الصفوف:
     ```javascript
     class Animal {
       constructor(name) {
         this.name = name;
       }
     }
     ```

5. **الوعود (Promises)**:
   - تسهيل التعامل مع العمليات غير المتزامنة، مما يساعد على كتابة شيفرة أكثر وضوحًا.
     ```javascript
     const fetchData = () => {
       return new Promise((resolve, reject) => {
         // عملية جلب البيانات
       });
     };
     ```

6. **الوحدات (Modules)**:
   - إمكانية تقسيم الشيفرة البرمجية إلى وحدات قابلة لإعادة الاستخدام:
     ```javascript
     export class MyClass { }
     import { MyClass } from './myModule.js';
     ```

### خلاصة

ES6 أدى إلى تحسين كبير في جافا سكريبت، مما جعلها أكثر قوة ومرونة، وساهم في تطوير تطبيقات ويب حديثة ومعقدة بشكل أسهل.
