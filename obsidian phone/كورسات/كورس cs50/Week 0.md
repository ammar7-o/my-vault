الأسبوع 0 (Week 0) - CS50
البت (Bit)
 * البت هو الوحدة الأساسية للمعلومات في الحوسبة.
 * يمكن أن يكون البت إما 0 أو 1.
 * 00 يمثل 2 بت.
 * 101 يمثل 3 بت.
 *    * * **مصباح مضيء** يمثل `1` (تشغيل).

     * مصباح مطفأ يمثل 0 (إيقاف).
تمثيل الأرقام الثنائية
يوضح هذا الجزء كيفية تمثيل الأرقام باستخدام النظام الثنائي (البتات).
 * يتم استخدام قوى العدد 2: 2^0, 2^1, 2^2, 2^3, 2^4, 2^5, 2^6, 2^7.
 * القيم المقابلة لهذه القوى هي: 1, 2, 4, 8, 16, 32, 64, 128.
 * مثال:
   * الرقم 65 ممثل ثنائيًا بـ 01000001 (أو 1000001 إذا تم حذف الصفر الأول).
     * 0 \\times 128 + 1 \\times 64 + 0 \\times 32 + 0 \\times 16 + 0 \\times 8 + 0 \\times 4 + 0 \\times 2 + 1 \\times 1 = 64 + 1 = 65.
   * الرقم 70 ممثل ثنائيًا بـ 01000110 (أو 1000110).
     * 0 \\times 128 + 1 \\times 64 + 0 \\times 32 + 0 \\times 16 + 0 \\times 8 + 1 \\times 4 + 1 \\times 2 + 0 \\times 1 = 64 + 4 + 2 = 70.
ASCII (American Standard Code for Information Interchange)
 * ASCII هو معيار لترميز الأحرف.
 * كل حرف أو رمز يتم تمثيله برقم.
 * مثال: الحرف A يقابله الرقم 65 في جدول ASCII.
الألوان الرئيسية (RGB)
 * RGB يرمز إلى Red (أحمر)، Green (أخضر)، Blue (أزرق).
 * يتم استخدام هذه الألوان الأساسية لإنشاء جميع الألوان الأخرى من خلال دمجها بنسب مختلفة.
 * كل لون من الألوان الثلاثة يمثله 8 بت (من 0 إلى 255)، مما يتيح 256 درجة لكل لون.
البكسل (Pixel)
 * البكسل هو أصغر عنصر في الصورة الرقمية.
 * كل بكسل يتكون من 3 بايت (Byte)، أي 24 بت (لأن 1 بايت = 8 بت، و 3 بايت \\times 8 بت/بايت = 24 بت).
   * هذه الـ 24 بت تستخدم لتمثيل قيم RGB (8 بت للأحمر، 8 بت للأخضر، 8 بت للأزرق).
 * لعرض الصور والفيديوهات، يتم استخدام مجموعة من البكسلات.
الخوارزميات (Algorithms)
 * الخوارزمية هي مجموعة من الطرق المختلفة للوصول إلى نفس الشيء (حل مشكلة معينة).
 * تصف الخوارزميات خطوات محددة ومنطقية لحل مشكلة أو إنجاز مهمة.
الكود الزائف (Pseudocode)
 * الكود الزائف هو طريقة لكتابة البرامج بلغة بشرية ومفاهيمية.
 * يستخدم لوصف منطق الخوارزمية دون الالتزام بقواعد بناء جملة لغة برمجة معينة، مما يسهل فهمها وتخطيطها قبل كتابة الكود الفعلي.
